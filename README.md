# Mevzuat Asistanı Landing Page

Bu proje, "Mevzuat Asistanı" mobil uygulaması için hazırlanmış modern, hızlı ve responsive bir tanıtım sayfasıdır (Landing Page).

## Özellikler
- **Modern Tasarım:** Glassmorphism ve canlı renk gradientleriyle dikkat çekici arayüz.
- **Responsive:** Tüm mobil cihazlar ve tabletlerle uyumlu esnek grid yapısı.
- **Saf HTML/CSS:** Framework bağımlılığı gerektirmeyen (Vanilla), çok hızlı yüklenen temiz kod mimarisi.
- **FontAwesome ve Google Fonts:** Dünyaca kabul gören okunaklı fontlar ve ikonlar.

## Kurulum ve Kullanım (GitHub Pages İçin)
Bu sayfayı ücretsiz olarak GitHub Pages üzerinde yayınlamak çok kolaydır:

1. Bu klasör içindeki (`index.html`, `style.css` ve `assets` klasörü) tüm dosyaları yeni bir GitHub reposuna (örn: `mevzuatasistani-web`) yükleyin.
2. Repo ayarlarınıza (Settings) gidin.
3. Sol menüden **Pages** sekmesine tıklayın.
4. "Build and deployment" altındaki Source kısmından `Deploy from a branch`'i seçin.
5. Branch kısmında `main` (veya `master`) dalını ve `/ (root)` klasörünü seçip "Save" diyerek kaydedin.
6. Birkaç dakika içinde siteniz `https://kullaniciadiniz.github.io/mevzuatasistani-web` adresinde yayında olacaktır!

## İçerik Düzenleme
- Metinleri veya linkleri değiştirmek için sadece `index.html` dosyasını açıp ilgili alanları düzenlemeniz yeterlidir.
- Ekran görüntülerini (`mockup.png` ve `dashboard.png`) değiştirmek için `assets` klasörü içindeki dosyaları kendi uygulamanızın gerçek ekran görüntüleriyle değiştirmeniz tavsiye edilir. Mevcut haliyle logonuzu demo olarak kullanır.
